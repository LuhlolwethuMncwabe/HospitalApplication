Hospital Management App
This is a simple hospital management system built using ASP.NET Core and Entity Framework Core. 
The project focuses on managing doctors and patients,and how they relate to each other in a database.
The app uses a code-first approach, where the models define the structure of the database. 
A doctor can have multiple patients, while each patient is assigned to one doctor. 
Basic validation is handled using data annotations to make sure the data entered is correct.
Overall, this project was mainly about understanding how to structure models, set up relationships, 
and work with a database using Entity Framework Core.
